# COD4 Game Types

Open console and run:

```/g_gametype {gametype}```

Example: ```/g_gametype sd```


Game Types:

* **Free For All** dm

* **Domination** dom

* **Headquarters** koth

* **Sabotage** sab

* **Search and Destroy** sd

* **Team Deathmatch** tdm

**Remember** that you must restart match after changing game mode for changes to take effect

```/map_restart```
